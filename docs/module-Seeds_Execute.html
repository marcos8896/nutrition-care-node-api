Seeds/Execute

Creates fake data based on the seed JSON files that were created by the prepare-seeds.js script. When a user runs this script, the user has to enter the wanted Model to seed and the number of wanted records. This file uses the faker.js library to generate all the dummy data.

Source:

Author:
  • Marcos Barrera del Río
Example
npm run execute:seeds Provider 10

(inner) typeOfSeedToGenerate(relationsTypes) → {string}

Source:
  • dev/seeds/execute-seeds.js, line 159

Determines what kind of seed is going to need the current model. There are two kind of seeds:

  • The simple, which only runs a function that create fake data by using the seed JSON file for the given seed model.
  • The complex, which also create fake models for the given seed model, however, this one also takes in consideration every related models by using the relations of the current model.
Parameters:
Name Type Description
relationsTypes Array.<string>

An array with all the relations from the current model.
Returns:

Returns the kind of seed that is going to be needed for the given seed model.
